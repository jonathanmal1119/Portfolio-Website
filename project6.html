Strands

Overview

Strands is a comprehensive full-stack salon booking website that provides a complete solution for salon management, client booking, and business operations. The platform serves clients, stylists, and salon owners with role-based dashboards and advanced analytics capabilities.

Technologies Used

  • Express.js for scalable backend architecture
  • MySQL for robust database management
  • Amazon S3 for cloud storage and file management
  • AWS EC2 for cloud hosting and deployment
  • RESTful API architecture
  • Authentication and authorization systems

Key Features

The platform includes comprehensive features for all user types:

  • Appointment scheduling system for clients
  • Service browsing and selection
  • Business profiles and management
  • Role-based dashboards (clients, stylists, owners)
  • Website analytics and metrics tracking
  • Photo uploading and management
  • Full shopping features including product CRUD operations
  • User cart management and purchase processing
  • User and endpoint authentication
