Coursify

Overview

Coursify is a full-stack web application built with React.js, Node.js, and MySQL to help students plan valid course schedules. The application features AWS-hosted database infrastructure and integrates a custom OpenAI assistant to suggest complete semester plans, significantly reducing planning time.

Technologies Used

  • React.js for dynamic frontend development
  • Node.js and Express.js for backend services
  • MySQL database for course data management
  • AWS for cloud hosting and database services
  • OpenAI API for intelligent course planning assistance
  • JavaScript for prerequisite and requirement logic

Key Features

This application offers comprehensive course planning capabilities:

  • Automated course schedule validation and planning
  • Prerequisite checking for 200+ courses
  • Major requirements and elective rule handling
  • AI-powered semester plan suggestions
  • AWS-hosted MySQL database for optimal performance
  • Responsive design for seamless user experience

Development Process

Built the complete full-stack application, migrating course data to AWS-hosted MySQL which improved performance by 40% and reduced costs by 95%. Developed sophisticated JavaScript logic to handle complex prerequisite chains and major requirements for over 200 courses.

Impact

The integration of the custom OpenAI assistant reduced course planning time by 60%, making it significantly easier for students to create valid schedules that meet all academic requirements. The AWS migration resulted in substantial cost savings and improved performance.
