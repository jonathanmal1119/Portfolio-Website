Metroid NES Game Recreation

Overview

Metroid NES Game Recreation is a faithful recreation of the classic 1986 Metroid game built using the Unity game engine. This project demonstrates advanced game development skills by recreating one of the most influential side-scrolling action-adventure games in gaming history, featuring Samus Aran's exploration of the planet Zebes.

Technologies Used

  • Unity Game Engine for complete game development platform
  • C# for game logic, player controls, and enemy AI
  • Unity 2D tools for sprite animation and level design

Development Process & My Contributions

My contributions to the Metroid NES Game Recreation project included:

Core Player Mechanics

  • Running movement system
  • Standing and running jump mechanics
  • Health and damage system
  • Morph Ball power-up
  • Missile weapon system
  • Health and missile collectables

Enemy Implementation

  • Assisted on Zoomer enemies (wall-crawling)
  • Assisted on Skree enemies (ceiling-diving)
  • Assisted on Ripper enemies (flying)
  • Zeb enemies (pipe-emerging)

Environment & Level Design

  • Door systems and room transitions
  • Missile-required doors
  • Lava blocks
  • Enemy and item placement

Audio & Deployment

  • Audio system implementation
  • Custom mechanics and levels

Technical Highlights

This project showcases expertise in game development, demonstrating proficiency in Unity, C# scripting, 2D game mechanics, and game design principles. The recreation maintains the challenging gameplay and exploration elements that made the original Metroid a timeless classic while leveraging modern development tools and techniques.
